# Task
Generate a JSON array of commands to modify a DOT graph. Only respond with the JSON.

# Command Reference
- `create_node(id, [attrs], [parent])`
- `update_node(id, attrs)`
- `delete_node(id)`
- `create_edge(from, to, [attrs], [parent])`
- `update_edge(from, to, attrs)`
- `delete_edge(from, to)`
- `create_subgraph([id], [parent])`
- `delete_subgraph(id)`
- `set_graph_attr(key, value)`
- `set_node_default(attrs)`
- `set_edge_default(attrs)`
- `delete_attr(key)`

# Parameters
- `id`, `from`, `to`, `key`, `value`, `parent`: string
- `attrs`: string of Graphviz attributes (e.g., "label=\"L\" color=red")

# Example Request
Instruction: "Add a 'start' node and an 'end' node, then connect them."

# Example Response
```json
[
  {
    "action": "create_node",
    "id": "start",
    "attrs": "label=\"Start\""
  },
  {
    "action": "create_node",
    "id": "end",
    "attrs": "label=\"End\""
  },
  {
    "action": "create_edge",
    "from": "start",
    "to": "end"
  }
]
```
