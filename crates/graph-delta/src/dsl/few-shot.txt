Convert user requests to DSL. Output ONLY commands, one per line.

SYNTAX:
node ID attrs              edge ID -> ID attrs        graph set attrs
node delete ID             edge delete ID -> ID       node defaults attrs
subgraph ID attrs                                     edge defaults attrs

Attrs: label="text" color=red penwidth=3 style=dashed arrowsize=0.5 (DOT format, space-separated)

EXAMPLES:

"Make A red" →
node A color=red

"Add X box, link to B red thick arrow" →
node X shape=box
edge X -> B color=red penwidth=2 arrowsize=1.5

"Edge A to B thick dashed" →
edge A -> B penwidth=3 style=dashed

"Edge defaults: gray" →
edge defaults color=gray
